# <img src="client/public/spendzy.png" width="25"> Spendzy — Full Stack Expense Tracker

Spendzy is a full-stack personal finance web application that helps users track income and expenses, understand spending habits, and visualize financial trends through interactive charts.

The application provides secure authentication, a responsive dashboard, and financial analytics to help users manage their money efficiently.

---

## 🚀 Live Demo

https://spendzy-app-client.vercel.app/

---

## ✨ Features

- User registration and login
- Secure authentication using JWT
- Add income and expense transactions
- Delete transactions
- Dashboard overview of total balance
- Category-wise expense analytics
- Monthly spending trends with charts
- Responsive UI built with Tailwind CSS

---

## 🛠 Tech Stack

**Frontend**
- React
- Vite
- Tailwind CSS
- Recharts

**Backend**
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose

**Authentication**
- JSON Web Token (JWT)

**Deployment**
- Frontend: Vercel
- Backend: Render
- Database: MongoDB Atlas

---

## 🏗 Architecture

Users  
↓  
React Frontend (Vercel)  
↓  
Node.js + Express Backend (Render)  
↓  
MongoDB Atlas Database
